# $$$Ax = b$$$
#### 首先是消元，此步骤与解 $$$Ax = 0$$$ 相同
- 若零行所对应的b的分量不为0，该方程无解

	- 可以理解为b向量超出了由A的线性组合所构成的向量空间，所以b不能用A的线性组合表示，故方程无解

- 若消元后主行个数(非零行)少于未知数个数，那么解是无穷多个的，即 b 可以被 A 的线性组合表示，同时b属于A的列空间
	- 如何快速找到一个特解：
		- 令所有自由变量为 0 ，易求得主变量的值，从而求出特解
	- 所有解的形式为 Ax = 0 的通解 + Ax = b的特解
- 当 r= n < m 要使Ax=b,b≠0有非零解，b必须取A中各列的线性组合；若b=0，方程只有零解(零向量)
- 当 r = n < m,每行都有主元，消元时不会有零行，故方程必有解，自由变量个数为 n - m = n - r个
- 当 r = m = n，方程必有解，方程 Ax = 0只有零解,Ax = b有唯一解
- 当 r < m , r < n，可能有0解或无穷解

# 四个子空间
#### 列空间C(A)
- 以列向量为基生成的空间
- $$$列空间在R^m内$$$

#### 零空间N(A)
- Ax = 0 的解 为零空间
- $$$零空间在R^n里$$$


#### 行空间/ $$$A^T的列空间,C(A^T)$$$
- 以行向量为基生成的空间
- $$$C(A^T)在R^n内 $$$


#### 转置零空间/左零空间 ,$$$ N(A^T)$$$
- $$$ A^Tx = 0的解$$$
- $$$N(A^T)在R^m内$$$

#### 性质
- 行空间和列空间的都是同维的,维数为r(rank秩)
- 零空间的维数为 **n - r** 维的
- 左零空间的维数为 **m - r**
- 行空间和零空间正交
- 列空间和左零空间正交
 




# 子空间投影

- $$$点p是点b在\overrightarrow{a}上的投影$$$
- $$$令\overrightarrow{e} = b - p ，\overrightarrow{e} 与\overrightarrow{a}垂直，把e称为误差，显然有 a^Te=0(这里用矩阵表示向量a和向量e)$$$
- $$$其中又有p = xa，因为p在a上，所以有一个x倍的关系$$$
- $$$a^Te=a^T(b-p)=a^T(b- xa) = 0$$$
- $$$化简得x = \frac{a^Tb}{a^Ta} ,代入p = xa ,得 p = a\frac{a^Tb}{a^Ta}$$$
- 因此可知b在a上的投影和a的无关和b有关，
- $$$Prj^b_a = Pb , P = \frac{a^Ta}{a^Ta},P为投影矩阵$$$

- 投影矩阵P满足$$$P = P^T = P^2$$$

## 投影的用处
$$$Ax = b不总有解，当b不在A的列空间内时，方程是无解的，但是我门可以把b投影到A的列空间，从而求出一个最接近解的\hat{x}$$$

- 首先我们知道A表示了一个列空间，当Ax= b无解时，b不在A的列空间内，但b在A的投影为p令e = b - p ，e为误差向量，e 与整个列空间垂直，故有方程(2)
$$
	A= \left[\begin{matrix}
	.&.&...&.\\
	a_1&a_2&...&a_n\\
	.&.&...&.
	\end{matrix}\right]\\
	\left[\begin{matrix}
		a_1^T\\
		a_2^T\\
		...\\
		a_n^T
	\end{matrix}\right]
	\left[\begin{matrix}
	b-A\hat{x}
	\end{matrix}\right] =	A^T \left[\begin{matrix}
	b-A\hat{x}
	\end{matrix}\right]=
	\left[\begin{matrix}
	0\\
	0
	\end{matrix}\right] (2)
$$



- $$$e在A^T的空间内，所以A^T的零空间和A的列空间正交$$$

- $$$ 方程可以改写成 A^TA\hat{x} = A^Tb,\hat{x} = (A^TA)^{-1}A^Tb$$$
- $$$ p = A \hat{x} = A(A^TA)^{-1}A^Tb = A,P = A(A^TA)^{-1}A^T$$$
- $$$如果A的各列线性无关A^TA可逆，同时这也是最小二乘法成立的大前提$$$
- $$$要证明A^TA可逆，等价证明A^TAx=$$$
	 1. $$$等死两边乘上x^T,$$$
	 2. $$$适当加括号得(Ax)^T(Ax)=0可看出Ax长度为零,因此得Ax=0，故x = 0$$$

- 列向量互相垂直必定线性无关，则是如果同时满足列向量相互垂直且各列向量都是单位向量，则可以说该矩阵是正交矩阵。

## 应用-最小二乘法
- 以找到最优直线为例，有些情况下无法找到一条直线通过所有的点，但是可以找到一条最优的线使得误差最小
 	$$$设直线为y=C+Dx误差向量e = |b-p|,误差的平方和为e^2 = e_1^2+e_2^2+e_3^2 对C、D求偏导数分别等于0，得方程组$$$
	$$$或者解方程A^TAx=A^T同样可以的到同样的方程组b$$$


# 特征值与特征向量
- 一个矩阵乘以一个向量如果结果是与该向量平行的向量，则结果与该向量的比值λ为该矩阵的特征值
$$
Ax=λx
$$

- 如果矩阵是奇异的，显然他有一个特征值λ = 0

- 投影矩阵上说表示的空间内的所有向量都是特征向量，且 特征值λ = 1

- n阶矩阵有n个特征值

- 特征值的和等于对角线元素的和

- 特征值之积为行列式的值

- 解下方程可以解出特征值

$$
	(A- λI)x=0,(此式子用作求特征向量)\\
	令det(A- λI)=0(解出λ逐一回代上式)
$$

- 特征向量是对角线元素减去特征值后解出来的零空间

- 对角线同时加上k特征向量不变,每个特征值数值加上k

- 特征向量可能是复数

- 行列式为零的矩阵,它的特征值会重复,且同一个特征值自由一个特征向量
